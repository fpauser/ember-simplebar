ember-simplebar
==============================================================================

A scrollbar component that wraps [SimpleBar](https://github.com/Grsmto/simplebar).



Compatibility
------------------------------------------------------------------------------

* Ember.js v3.12 or above
* Ember CLI v2.13 or above
* Node.js v10 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-simplebar
```


Usage
------------------------------------------------------------------------------

```
<SimpleBar>
  ...
</SimpleBar>
```
